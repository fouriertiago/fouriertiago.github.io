Système de Monitoring Intelligent

Détails de l'architecture backend mise au point pour l'interconnexion entre Zabbix et le LLM.

API Python Webhook

Pour faire communiquer le serveur de supervision Zabbix avec l'IA Flowise, j'ai développé un script middleware en Python utilisant le framework Flask.

Langage & Outils
Python 3.11 / Flask / Requests
Temps de réponse
< 1.2 seconde via API
Sécurité
Bearer Token Auth
Méthodes
POST Webhook Payload
app.py — Flask Middleware
from flask import Flask, request
import requests

app = Flask(__name__)
FLOWISE_URL = "http://localhost:3000/api/v1/prediction/..."

@app.route('/webhook', methods=['POST'])
def handle_zabbix_alert():
    data = request.json
    trigger_name = data.get('trigger')
    host = data.get('host')
    
    # Prompt envoyé à l'IA pour diagnostic
    prompt = f"Analyse l'alerte {trigger_name} sur {host}"
    
    response = requests.post(FLOWISE_URL, json={"question": prompt})
    return {"status": "processed", "ia_reply": response.json()}, 200

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
